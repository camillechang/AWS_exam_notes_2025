**QuickSight SPICE vs Direct Query**,
| 特性    | SPICE        | Direct Query      |
| ----- | ------------ | ----------------- |
| 性能    | 快（内存缓存，列式存储） | 取决于数据源（可能慢）       |
| 数据实时性 | 延迟（取决于刷新频率）  | 实时                |
| 数据库负载 | 低            | 高（每次查询都会 hit 数据库） |
| 成本    | 按 SPICE 容量收费 | 可能有数据扫描/数据库查询费用   |
| 使用场景  | 高频查询、多人访问    | 小数据集、实时性强的场景      |
